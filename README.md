# 📰 Fake Headline Generator (Python)

A fun Python project that generates random, humorous, and creative fake news headlines using predefined templates and word lists.

---

## 🚀 Features

- 🎲 Generate random fake headlines instantly  
- 🧠 Uses customizable word lists and templates  
- ⚡ Lightweight and fast  
- 🛠️ Beginner-friendly Python project  
- 🔁 Infinite combinations of headlines  

---

## 🛠️ Tech Stack

- Python 3.x

  ---

## ⚙️ How It Works

The application generates headlines by randomly combining:

- **Subjects** (e.g., Scientists, Celebrities, Government)  
- **Actions** (e.g., discover, ban, reveal)  
- **Objects** (e.g., time travel, secret cities, invisible cars)  

Each time the user clicks a button, a new headline is created.
